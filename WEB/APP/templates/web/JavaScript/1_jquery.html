선택자

선택자는 HTML 요소를 선택하여 가져온다.

기본 선택자
직접 선택자, 근접 선택자

탐색 선택자
위치 선택자, 속성 선택자

객체 조작
선택한 요소 삭제, 생성, 복제, 속성 변경

            $(function){
                $("요소 선택").css("속성","값");
                $(".wrap_1").css("border","1px solid orange");
            });

            DOM (Document Object Model) : HTML 문서 객체 구조


            //1. 선택자
$(function(){
    // 직접 선택자
    $(".wrap_1").css("border","1px solid orange");          // class가 wrap_1인 div태그를 선택해서 css 적용
    // 근접 선택자
    $(".wrap_1 p").css("background-color","yellow");        // wrap_1의 자식태그인 p를 선택해서 노란색으로 표시
    $(".active").next("p").css("background-color","aqua");  // class가 active인요소의 다음에 나오는 요소인 p를 선택
    // 위치 선택자
    $(".wrap_2 p").eq(2).css("background-color","red");     // p태그중 index가 2번인 p태그에 적용
    // 속성 선택자
    $("input[type=text]").css({"background-color":"orange"});     // input태그중 type이 text인 태그를 선택
    // 객체 조작
    $(".wrap_1 p").eq(1).removeClass("active");             // active 클래스 삭제
    $(".wrap_1").append("
append tag test
");          // p태그 삽입
});
